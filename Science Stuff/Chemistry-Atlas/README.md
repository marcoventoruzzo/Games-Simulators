# Chemistry Atlas

An interactive, English-language chemistry learning tool built with Python and Streamlit.

## Modules

1. Elements — complete interactive periodic table and shell model
2. Molecules — 61 common substances with schematic molecular or formula-unit models
3. Reactions — four reaction families with animated examples
4. Balancing — guided conservation-of-mass exercises
5. Language & Measure — moles, formulae, and nomenclature
6. Lab — seven classroom experiments with safety guidance

## Run locally

```bash
python -m pip install -r requirements.txt
streamlit run streamlit_app.py
```

For Streamlit Community Cloud, select `streamlit_app.py` as the entry point.
